# Victor Omitoyin — PM Portfolio

A project management portfolio built with HTML and hosted on GitHub Pages.

## Structure
- `index.html` — main portfolio page
- `projects/` — individual project case study pages

## Adding a new project
1. Open `index.html` and find the section marked "YOUR PROJECTS LIVE HERE"
2. Copy one project block and fill in your details
3. Create a matching page inside the `projects/` folder
